# ScriptLoader
## Purpose
Tiny (goal: ~2kb gzip), inline script loader to load other scripts with:
- maximum backwards compatibility
- conditional loading
- failover
- retry with exponential backoff policy
- inter-script dependency evaluation
- automatic polyfills
